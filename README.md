# XGON

Sistema de gestão condominial desenvolvido com Flutter e Firebase.

## Tecnologias

- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Firebase Messaging

## Funcionalidades

- Gestão de moradores
- Gestão de portaria
- Agenda de eventos
- Ocorrências
- Controle financeiro
- Upload de documentos
- Assistente virtual Axia
